# FindTheBug

FindTheBug is a windows desktop game made with python where you have to find the bugs.

<img width="589" height="499" alt="Screenshot 2026-06-26 125623" src="https://github.com/user-attachments/assets/7e4690b5-5338-48d3-b9ac-a14cc3b9980e" />

## Features
- 10 levels with different apps
- Find bugs by clicking and typing
- 3 different difficulties - Easy, Medium, Hard
- Different type of apps
- Total score

## How to play
- Click on broken elements in some app
- Type whats wrong in the app
- Skip if the level feels to hard

## How to download
1. Download the .exe file from releases
2. Open it and play
If Windows SmartScreen blocks it then click "More info" and then "Run anyway"

## Tech stack
Python
Tkinter
